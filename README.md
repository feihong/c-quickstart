# Feihong's C quickstart

## Installation

    brew install nushell
    just install

## Commands

Build a program and run a program

    clang -Wall hello.c && ./a.out

Start C programming environment (includes useful aliases defined in `env.nu`)

    just env

Build `hello` program

    make hello
