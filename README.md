# Feihong's C quickstart

## Installation

Install [XCode](https://apps.apple.com/us/app/xcode/id497799835?mt=12).

## Commands

Build a program

    cc hello.c -o hello

Run the program you built

    ./hello

Build `hello` program

    make hello
